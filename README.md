#Voyage
A travel booking application for students

##Features
-Search bar for destinations
-Category based browse for flights, rail pass, hostels
-Creates Itinerary for various locations
-Section to read stories and reviews from other travellers
-Sign up with email for exclusive student offer alerts

##Tech Stack
-Next.js (React framework)
-Deployed on Vercel

## User Stories 
-As a student, I want to explore different destinations so I can travel to suitable places
-As a student, I want to browse different travel categories so I can find suitable flights, rails, hostel
-As a student I want to create different itineraries so I can organise and optimize my trips
-As a student I want to browse curated travel spots so I can discover popular tourist destinations
-As a student I want to sign up with my email so I can receive exclusive student offers for my trips
