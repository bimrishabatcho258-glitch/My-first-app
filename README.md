# ScholasticEditorial
A task management app for students to organize assignments and track deadlines.

## Target Users  
Students managing coursework and deadlines.

## Features  
- Create tasks with title, due date, and priority  
- View daily agenda and upcoming tasks  
- Mark tasks as complete  
- Track weekly progress  
- View tasks by subject  

## User Stories  

- As a student, I want to create tasks, so that I can track my assignments.  
- As a student, I want to view my daily agenda, so that I know what to do today.  
- As a student, I want to mark tasks as complete, so that I can track my progress.  
- As a student, I want to see my weekly progress, so that I stay motivated.  
- As a student, I want to view upcoming tasks, so that I don’t miss deadlines.  

## Tech Stack  
- Next.js (React framework)  
- Deployed on Vercel
